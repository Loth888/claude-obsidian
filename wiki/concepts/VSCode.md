# VSCode

## 定义

VSCode（Visual Studio Code）是微软开发的**免费开源代码编辑器**，支持Windows、macOS、Linux全平台。它以轻量、快速、插件丰富著称，是目前最流行的代码编辑器之一。

## 核心要点

### 1. 为什么选择VSCode？

| 优势 | 说明 |
|------|------|
| **免费** | 完全免费使用 |
| **全平台** | Windows/Mac/Linux都支持 |
| **轻量快速** | 启动快，占用资源少 |
| **插件丰富** | 海量插件扩展功能 |
| **AI编程友好** | 支持各种AI编程插件 |

### 2. 核心界面元素

打开VSCode后，左侧边栏有几个关键图标：
- **📁 资源管理器** - 管理文件和文件夹
- **🔍 搜索** - 全局搜索代码
- **🌿 源代码管理** - Git版本控制
- **🧩 插件市场** - 安装扩展插件
- **🐛 调试** - 运行和调试程序

### 3. AI编程必备插件

| 插件 | 用途 |
|------|------|
| **Kimi Code** | 国产AI编程助手 |
| **Codex** | OpenAI官方插件 |
| **Gemini** | Google AI编程插件 |
| **Remote - SSH** | 远程连接云服务器 |

### 4. 基本操作

**打开文件夹**：
1. File → Open Folder
2. 选择项目文件夹
3. 整个项目的文件都会显示在左侧

**安装插件**：
1. 点击左侧"四个小方块"图标（插件市场）
2. 搜索插件名称
3. 点击安装
