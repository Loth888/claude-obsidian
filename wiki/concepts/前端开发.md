# 前端开发

## 定义

前端开发是构建**用户直接看到和交互的界面**的技术领域。它负责网页或应用的外观、布局、交互逻辑，是用户与系统之间的桥梁。

刘润的比喻：
> "前端就像饭店的前厅，是用户能看到、能操作的东西。比如网页、按钮、输入框。"

## 核心要点

### 1. 前端 vs 后端

| 维度 | 前端 | 后端 |
|------|------|------|
| **可见性** | ✅ 用户可见 | ❌ 用户不可见 |
| **运行位置** | 浏览器/客户端 | 服务器 |
| **核心技术** | HTML、CSS、JavaScript | Python、Node.js、数据库 |
| **主要职责** | 界面展示、用户交互 | 数据处理、业务逻辑 |
| **比喻** | 饭店前厅 | 饭店后厨 |

### 2. 前端的核心技术

| 技术 | 作用 | 说明 |
|------|------|------|
| **HTML** | 结构 | 定义页面内容和结构 |
| **CSS** | 样式 | 控制页面外观和布局 |
| **JavaScript** | 交互 | 实现动态功能和用户交互 |

### 3. AI编程中的前端

在AI编程入门流程中：
- AI生成的前端代码 = **用户界面**
- 包含：输入框、按钮、表格、图表等
- 用户直接操作和看到结果

**示例**：客户评分工具的前端
- 输入框：输入客户信息
- 按钮：提交分析
- 表格：展示评分结果
- 图表：可视化数据

### 4. 浏览器的开发者工具

**打开方式**：右键 → 检查（或按F12）

**常用功能**：
- **Elements**：查看和修改HTML/CSS
- **Console**：查看错误信息（Debug用）
- **Network**：查看网络请求
- **Sources**：调试JavaScript代码

**Debug技巧**：
> "打开'console'选项卡，如果有红色报错的代码，就可以直接发给AI对话框，让它修复。"

## 实践应用

### 场景：客户评分工具前端

**界面元素**：
1. **输入区** - 客户信息表单
2. **上传区** - Excel文件上传
3. **按钮** - 开始分析
4. **结果区** - 评分表格展示
5. **导出** - 下载结果

**AI生成代码示例**：
```html
<!-- 输入表单 -->
<input type="text" placeholder="客户名称">
<input type="text" placeholder="联系方式">

<!-- 上传按钮 -->
<input type="file" accept=".xlsx">

<!-- 分析按钮 -->
<button onclick="analyze()">开始分析</button>

<!-- 结果表格 -->
<table id="results">
  <tr><th>客户</th><th>评分</th><th>建议</th></tr>
</table>
```

## 局限与解决

### 前端无法直接调用API的问题

**原因**：浏览器环境不安全
- API Key会暴露在代码中
- 任何人打开开发者工具都能看到

**解决**：后端代理
- 前端 → 发送请求到后端
- 后端 → 调用API（Key安全存储）
- 后端 → 返回结果给前端
