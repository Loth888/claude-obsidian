# 个人知识管理

## 定义
个人知识管理（Personal Knowledge Management, PKM）是指个人对知识的获取、整理、存储、加工和输出的系统化流程，旨在构建可长期调用和复用的知识体系。

## 核心目标

个人知识管理对于以下四类人群至关重要：

1. **希望成为专家**：在特定领域/话题上不断精进，成为专家
2. **希望超强IPO**：构建自己的IPO模型，大量阅读、消化内容
3. **希望持续输出**：拥有持续输出能力，如持续写作、写脚本、做课、做创意
4. **希望长期变强**：以10年为单位，打造某些能力超强的自己

## 三大核心困扰

### 困扰1：缺少审美，没见过好的
- **症状**：特别渴望做知识管理，但没见过真正好的案例
- **原因**：知识管理是比较私密的事情，很难看到别人的笔记本具体怎么管理
- **解决**：学习优秀案例，建立知识管理审美

### 困扰2：信息量太大，大量丢失
- **症状**：平时接收的信息量太大，大量文章、会议、课程、图片、聊天涌来
- **问题**：感觉里面有很多精华，但没有趁手的工具可以记下来
- **解决**：建立高效的信息捕获系统

### 困扰3：收藏很多，用得很少
- **症状**：平时收藏大量文章，写大量碎片笔记，但一直没什么用
- **后果**：积累越多、内容越乱、压力越大，反而不想要了
- **数据**：可能做了100件事，只沉淀下来80%，最后能长期调用的知识只有1%

## 关键能力

| 能力 | 说明 |
|------|------|
| **信息捕获** | 快速记录有价值的信息 |
| **知识整理** | 结构化组织和分类 |
| **知识加工** | 将信息转化为可理解的知识 |
| **知识输出** | 将知识转化为作品 |
| **知识复用** | 长期存储和快速检索 |
