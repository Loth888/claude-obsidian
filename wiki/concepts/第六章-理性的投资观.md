# 第六章：理性的投资观

## 概述
本章探讨正确的金钱观和投资理念，帮助建立理性的投资观。

## 核心观点

### 29. 金钱观
- **金钱的本质**：金钱是手段而非目的
- **正确金钱观**：
  1. 钱是挣出来的，不是省出来的
  2. 钱是花出去的，用来买经历和服务
  3. 钱是投资出去的，用来换取更多钱
- **三个花钱原则**：
  - 投资自己
  - 购买经历
  - 适度享受

### 30. 风险意识
- **风险类型**：
  - 市场风险
  - 个人风险（失业、疾病）
  - 通胀风险
- **风险管理**：
  1. 分散投资
  2. 保持流动性
  3. 购买保险
  4. 持续收入能力

### 31. 投资
- **投资原则**：
  1. 永远不要借钱投资
  2. 投资指数基金
  3. 长期持有
  4. 定期定额
- **不投资的东西**：
  - 不熟悉的领域
  - 炒作概念
  - 高杠杆产品

### 32. 投资中的误区
- **常见误区**：
  1. 试图低买高卖
  2. 过度自信
  3. 追求短期收益
  4. 把运气当能力
- **正确认知**：
  - 市场难以预测
  - 分散风险
  - 长期投资

### 33. 围绕投资目的进行资产配置
- **投资目的**：
  - 养老
  - 子女教育
  - 财务自由
- **资产配置原则**：
  1. 根据目的配置
  2. 根据时间配置
  3. 定期调整
- **建议配置**：
  - 短期：现金、货币基金
  - 中期：债券
  - 长期：股票指数基金

## 关键启发
1. 金钱是手段不是目的
2. 风险意识是投资的前提
3. 避免投资误区
4. 根据目的进行资产配置
