# 代码编辑器

## 定义

代码编辑器（Code Editor）是开发者用来**编写、编辑、检查和管理代码**的软件工具。它提供语法高亮、代码补全、错误检查等功能，是编程工作的"工作室"。

## 核心要点

### 1. 为什么需要代码编辑器？

虽然代码本质上是文本，但写程序没那么简单：
- 需要配置**语言环境**
- 遵循严格的**文件结构**
- **引用大量文件**和依赖
- 需要**语法检查**和错误提示

代码编辑器就是帮开发者处理这些复杂性的工具。

### 2. 核心功能

| 功能 | 说明 |
|------|------|
| 语法高亮 | 不同代码元素用不同颜色显示 |
| 代码补全 | 自动提示和补全代码 |
| 错误检查 | 实时发现语法错误 |
| 文件管理 | 管理项目中的所有文件 |
| 插件扩展 | 安装各种功能插件 |
| 调试工具 | 运行和调试程序 |

### 3. 常见代码编辑器

| 编辑器 | 特点 | 适用场景 |
|--------|------|----------|
| **VSCode** | 免费、轻量、插件丰富 | 通用，AI编程首选 |
| JetBrains系列 | 专业、功能强大 | 专业开发 |
| Sublime Text | 极速、简洁 | 轻量级编辑 |
| Vim/Emacs | 高度可定制 | 极客首选 |

## 实践应用

### AI编程场景下的使用

在AI编程入门流程中：
1. **安装VSCode** - 建立编程工作室
2. **安装AI插件** - 把AI请进工作室
3. **打开工作文件夹** - 划定AI工作范围
4. **与AI对话** - 在编辑器里指挥AI写代码
